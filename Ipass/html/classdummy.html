Detailed Description

Dummy contains a copy function.

Copier is a function that creates a non const matrix(matrix_dummy) from data(data_input).
The advantage of this is that your orginal data won't be altert.
Example: You want to render Hello, but you want it rotated 90\370. if you send you orginal matrix, it will be rotated.
Then displayed, but then rotated again and then displayed. Copy would be send, rotated, displayed.
The a other copy will be made and send, rotated, displayed
