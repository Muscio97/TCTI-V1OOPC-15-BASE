Detailed Description

Command contains a function that sends data to the MAX7219 thru SPI.

Commander gets a 3d list( (*data)[8][2] ). Command uses 3 for loops to read out the list. First loop is with letter out of the list will be displayed.
Second loop is for getting the data per row.
Final loop is for the amount of matrices.
It makes it posseble to grap the last(or next) letter out of the data input.
