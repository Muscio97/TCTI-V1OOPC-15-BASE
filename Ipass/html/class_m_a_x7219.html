Detailed Description

This is a storage for:

This class holds all supported charactres, test modes, render, setiings and the sting to dislpay tekst switch.
The characters are all 8x8 (byte)matrices.
Test mode is a mode to test the matrix, it will display every currunt supported charchter. rotated and sometimes inverded.
Render makes a copy of the orignal 8x8 (byte)matrix, sends that to mbin_to_mhex. mbin_to_mhex will process that in a 8x2 (byte)matrix.
Render will take that 8x2 (byte)martix and put it in a ?x8x2 (byte)matrix. When all charachters are put in the ?x8x2 (byte)matrix
render will send it to command(). Settings is a ?x2 matrix of settings. The switch take a input string and with every supported it will call render with the there to belonning character.
