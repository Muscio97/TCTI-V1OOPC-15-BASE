Detailed Description

Rotate contains a rotator function.

Rotate is a function that rotates a matrix. The size of the matrix can devined wtih count.
Count is number of rows and columns. This is because of the way the fucntion works.
rotate_matrix() gives the to be rotated data to rotator()

Example: matrix is 4x4, count 4. It takes (0,0),(3,0),(3,3),(0,3). Then it will rewrite this to: (3,0),(3,3),(0,3),(0,0.
Now it is rotated.next it takes (1,0),(2,0),(3,2),(1,3). Then it will rewrite this to: (2,0),(3,2),(1,3),(1,0).
Eventually it will have rotated the whole matrix.

The matrix it self will be altered, so i advise you to send a copy of your matrix to it.
